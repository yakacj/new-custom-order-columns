== Description ==

This plugin for WordPress/WooCommerce stores, by this plugin you can add new unlimited custom order columns to shop order list at admin side. 
Need to enter column title and meta key for obtain/show value for new column in plugin settings page.

== Installation ==

1. Go to WP admin panel, click 'Plugins' -> 'Add new'. In the search input box, type 'new custom order columns'.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. OR, upload "new-order-columns-in-woocommerce.zip" to the "/wp-content/plugins/" directory.
4. Install and activate the plugin.
5. Plugin settings found beside activate/deactivate plugin on plugins page, also found on admin menu under woocommerce menu.
